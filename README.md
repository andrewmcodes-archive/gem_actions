## Installation

Adding to a gem:

```ruby
# my-cool-gem.gemspec
Gem::Specification.new do |spec|
  # ...
  spec.add_dependency "gem_actions"
  # ...
end
```

Or adding to your project:

```ruby
# Gemfile
gem "gem_actions"
```

### Supported Ruby versions

- Ruby (MRI) >= 2.5.0
